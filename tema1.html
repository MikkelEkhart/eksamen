Tema 1. Grundlæggende Web

Temaet

Vores første tema gik ud på at få udviklet et website, for aller første gang. Her startede vi med at lærer om det grundlæggende, som f.eks. hvor vigtigt det er at et webiste er responsivt, så det både fungerer på web samt mobil. Og vores første møde med grids i CSS.

Formål

Formålet med dette site var at få os sat ind i det mest basale CSS og HTML, for at kunne viderebygge på det. Vi skulle lave en brugervenlig site, ved brug af semantisk HTML og CSS grids.

Proces

Det blev brugt som studiestartsprøve - vi startede med at lave meget af siden sammen i klassen, og fik en god insigt i hvad der skulle laves. Jeg startede med at bygge min HTML struktur op, efterfulgt af en style.css fil, hvor jeg brugte grid til at få siden til at se mere brugervenlig ud.

Læring

Første tema lagde en rigtig god grundlinje for hvad vi kommer til at lærer. Vi har fået mange værktøjer, og forskellige sider vi kan benytte os af til at videreudvikle vores evner selv. Som f.eks. w3schools. Jeg lærte meget om brugervenlighed, designprocess, semantisk HTML og basal CSS med grids.

Refleksion

Jeg lærte meget om hvor vigtigt det var at siderne man kodede i HTML samt CSS, blev responsive på f.eks. tlf. eller computer. Første tema var alt i alt, en rigtig god trædesten - vi fik styr på basic HTML og CSS, og blev lært om vigtigheden af det tekniske og det visuelle.
